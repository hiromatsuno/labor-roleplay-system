# 労務相談ロールプレー訓練システム

人事労務担当者の対応力を向上させるための訓練アプリケーションです。

## 特徴

- **12種類の事前準備シナリオ**: メンタルヘルス、ハラスメント、労働時間など多様なケースを用意
- **カスタムシナリオ作成**: 独自の相談ケースを簡単に作成可能
- **ワンクリックコピー**: 生成されたプロンプトを簡単にコピーしてAIチャットで使用
- **カテゴリー分類**: 労務相談の種類ごとに整理されたシナリオ
- **直感的なUI**: 使いやすいインターフェースで効率的な訓練が可能

## インストール

```bash
pip install -r requirements.txt
```

## 使い方

### アプリケーションの起動

```bash
streamlit run labor_roleplay_app.py
```

### 基本的な使用手順

1. **シナリオ選択**: 
   - 「テンプレート選択」タブから既存のシナリオを選択
   - または「カスタム作成」タブで独自のシナリオを作成

2. **プロンプト生成**:
   - シナリオを選択すると自動的にプロンプトが生成されます

3. **AIチャットで実践**:
   - 「クリップボードにコピー」ボタンでプロンプトをコピー
   - Claude等のAIチャットに貼り付けて訓練を開始

4. **対話と評価**:
   - AIが相談者役となり、あなたは人事労務担当者として対応
   - 対話終了後、AIから法的・コミュニケーション面での評価を受けられます

## 用意されているシナリオカテゴリー

- **メンタルヘルス**: 復職相談、メンタル不調対応など
- **労働時間**: 長時間労働、残業問題など
- **ハラスメント**: パワハラ、セクハラ対応など
- **育児・介護**: 育休復職、時短勤務など
- **退職・解雇**: 退職勧奨、整理解雇など
- **賃金**: 未払い賃金、給与トラブルなど
- **人事異動**: 配置転換、出向など
- **採用・試用期間**: 試用期間満了、本採用拒否など
- **職場環境**: 人間関係トラブルなど
- **休暇・休業**: 有給休暇取得、特別休暇など
- **安全衛生**: 健康診断、安全配慮など

## プロンプトテンプレートのカスタマイズ

`prompt_templates.py`ファイルの`PROMPT_TEMPLATES`リストに新しいシナリオを追加できます:

```python
{
    "title": "シナリオのタイトル",
    "category": "カテゴリー名",
    "consultation": "相談内容",
    "background": "背景情報",
    "attitude": "相談者の態度"
}
```

## 技術スタック

- **Pyperclip**: クリップボード操作

## 注意事項

- このシステムは訓練用途です。実際の労務相談には専門家への相談をお勧めします
- 生成されるシナリオはフィクションです
- AIからの評価は参考情報として活用してください
